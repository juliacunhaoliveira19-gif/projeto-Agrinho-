# 🌾 Conexão Campo-Cidade: Divulgação de Agricultores Locais

Este é um projeto focado em aproximar os produtores rurais dos consumidores urbanos. A plataforma funciona como uma vitrine digital, permitindo que agricultores divulguem seus trabalhos e produtos, e que o público da cidade encontre contatos diretos (como WhatsApp, e-mail ou redes sociais) para apoiar e comprar diretamente de quem produz.

## 🚀 Sobre o Projeto

O grande objetivo deste site é encurtar a distância entre o campo e a cidade, promovendo o comércio justo, o consumo consciente e fortalecendo a agricultura familiar local. Com uma interface simples e intuitiva, os usuários podem navegar pelos perfis dos agricultores e, com apenas um clique, ser direcionados para os canais de atendimento de cada um.

## 🛠️ Tecnologias Utilizadas

Para o desenvolvimento deste projeto, foram utilizadas as tecnologias fundamentais da web:

* **HTML5:** Para a estruturação semântica do conteúdo e dos perfis.
* **CSS3:** Para a criação de um design responsivo, moderno e inspirado nas cores da natureza.
* **JavaScript:** Para a lógica de interatividade e o direcionamento dinâmico dos cliques de contato.
